# Portfolio Page Design Specification

## Overview
The Portfolio page (`portfolio.html`) features a unique entrance animation, project toggle system, accordion navigation, and horizontal scrolling case study pages.

---

## Color Theme
**Same Kenyan Flag Palette**:
- Black: `#000000`
- Red: `#ce1126`
- Green: `#006b3f`
- White: `#ffffff`
- Background: `hsl(48, 10%, 90%)`

---

## Typography
**Same Font System**:
- **Asimovian** - Display
- **Space Mono** - Monospace
- **Enriqueta** - Serif

## Entrance Animation

### SVG Effect
- **File**: `css/portfolio-entrance.css`, `js/portfolio-entrance.js`
- **Asset**: `images/PortFolioImageDesign.svg` [PLACEHOLDER]
- **Animation Sequence**:
  1. SVG graphic appears
  2. Morphs/transitions
  3. Fades out
  4. Portfolio content fades in
- **Duration**: ~2-3 seconds
- **Timing**: Runs once on page load

### FOUC Prevention
- **Style**: Portfolio content hidden until entrance complete
- **Class**: `.entrance-complete` added to body when done
- **Opacity**: 0 until animation finishes

## Project Toggle System

### Toggle Container
- **Class**: `.project-toggle-section`
- **Layout**: Horizontal flex, centered
- **Position**: Below header, above content

### Toggle Buttons
- **Count**: 3 projects
- **Structure**:
  ```html
  <button class="project-toggle-btn active" data-project="1">
    <span class="toggle-number">01</span>
    <span class="toggle-label">Eastleigh</span>
  </button>
  ```

2. **Delivah Dispatch Hub** (02)

   - Freight carrier management

3. **EduManage SMS** (03)
   - School management system

#### Button Styling
- **Active State**: Highlighted, larger
- **Inactive State**: Muted, smaller
- **Transition**: Smooth scale and color change
- **Number**: Large, bold
- **Label**: Smaller, below number

## Accordion Navigation

### Container
- **Class**: `.accordion-nav`
- **Position**: Fixed left sidebar
- **Width**: ~250-300px
- **Scroll**: Independent vertical scroll

### Accordion Items
- **Count**: 9 chapters per project
- **Structure**:
  ```html
  <div class="accordion-item active" data-chapter="details">
    <button class="accordion-header" aria-expanded="true">
      <div class="progress-circle">...</div>
      <span class="accordion-title">Project Details</span>
      <svg class="accordion-icon">...</svg>
    </button>
    <div class="accordion-content">
      <a href="#overview" class="accordion-link active">Overview</a>
      <a href="#role-timeline" class="accordion-link">Role & Timeline</a>
      <a href="#tech-stack" class="accordion-link">Tech Stack</a>
    </div>
  </div>
  ```

### Progress Circles
- **Size**: Small circular SVG
- **Fill**: Animates based on scroll progress
- **Percentage**: Displayed as text inside circle
- **Color**: Kenyan green when active

#### Circle Animation
- **Stroke**: Circular progress indicator
- **Dash Offset**: Animates from 88 (0%) to 0 (100%)
- **Timing**: Updates as user scrolls through pages

### Accordion Behavior
- **Click**: Expands/collapses content
- **Active State**: Expanded, highlighted
- **Icon**: Chevron rotates 90deg when expanded
- **Transition**: Smooth height animation

### Chapter Links
- **Display**: Nested under accordion header
- **Active State**: Bold, colored
- **Click**: Scrolls to corresponding page
- **Hover**: Subtle background color

## Responsive Behavior

### Desktop (> 1024px)
- **Layout**: Accordion sidebar + horizontal pages
- **Scroll**: Both vertical (accordion) and horizontal (pages)

### Tablet (768px - 1024px)
- **Accordion**: Collapsible, narrower
- **Pages**: Smaller padding

### Mobile (< 768px)
- **Accordion**: Top horizontal tabs or hamburger menu
- **Pages**: Vertical scroll instead of horizontal
- **Simplified**: Reduced complexity

## Notes for Recreation

1. **Entrance Animation**: SVG should be preloaded for smooth playback
2. **Horizontal Scroll**: Use CSS scroll-snap for page alignment
3. **Progress Circles**: Calculate based on scroll position within chapter
4. **Accordion State**: Persist active state during navigation
5. **Page Content**: Load from data structure for easy updates
6. **Smooth Scrolling**: Use `scrollIntoView({ behavior: 'smooth' })`
7. **Active States**: Sync between accordion, pages, and progress
8. **Mobile Layout**: Convert horizontal to vertical scroll
9. **Performance**: Limit DOM nodes, use virtual scrolling if needed
10. **Accessibility**: Ensure all interactions work with keyboard only
